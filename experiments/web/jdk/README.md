# JDK 8-12 Features Explorer

Preserved from the former `From jdk8 to jdk12` draft.

Files:

- `index.html`: standalone browser entrypoint
- `app.js`: React app that renders the JEP list
- `data/jdk-data.json`: generated dataset grouped by JDK version
- `styles.css`: light styling for the standalone page

Data refresh:

- `tools/jdk-crawler/` fetches OpenJDK and Jira data and rewrites `data/jdk-data.json`

Reference links kept from the draft:

- https://dzone.com/articles/4-new-jeps-proposed-for-java-12
- https://howtodoinjava.com/
